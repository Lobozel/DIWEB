Respuesta a Cómo se genera un sprite

Un sprite es un mapa de bits, generalmente compuesto por varias imagenes pequeñas y un fondo transparente o de color sólido.

Suelen utilizarse en videojuegos y su uso es similar al una carpeta de recursos de imágenes, es decir, podriamos tener un sprite de un personaje con distintas imagenes de este mismo en diferentes ángulos para posteriori crear el efecto de movimiento del personaje dentro del juego.

Existen programas para crear, manipular y cortar sprites, sin embargo, con cualquier editor de imágenes (como GIMP) podríamos hacerlo.

Tan solo necesitamos unir varias imágenes sin solaparlas en un mismo mapa de bits, a ser posible con fondo transparente y con alguna estructura que más adelante nos ayude a separar cada imagen para su uso.

Como he mencionado antes, existen programas para cortar sprites. Estos programas se utilizan para separar cada imagen contenida en el mapa de bits que representa el sprite en distintas imagenes JPEG o PNG.


Para usar un sprite en una página Web con CSS utilizamos las clases, creamos un bloque que represente y carge la imagen con '*' y una clase por cada imagen donde definamos su posición con background-position.

Existen muchas otras formas, pero esta es la que voy a utilizar yo de ejemplo.


Código:
*[sprite]{ background: url('../resources/img/sprite-dw.png') no-repeat top left; width: 127px; height: 195px; display: inline-block; }
.html{ background-position: 0px 0px; }
.css{ background-position: -171px 0px; }
.js{ background-position: -340px 0px; }
